# QA Pet Project

A learning-oriented pet project to build practical QA automation skills.

## Goal

Build a hands-on QA automation project that demonstrates:
- test design principles
- API testing
- integration testing with a database
- CI/CD pipeline setup
- E2E testing

The project is structured as a step-by-step learning journey over several months.

Planned tech stack:
- Python
- pytest
- FastAPI
- PostgreSQL
- Docker
- GitHub Actions
- Playwright
- Allure

## Current Status

- strings
